BotSpot footer-placement fixture

How to use:
  1. Open this file directly in a browser (no plugin needed).
  2. Open DevTools console.
  3. Paste the placement script (copy from includes/class-botdot-wp-content-injector.php → inject_placement_script()) into the console and press Enter.
  4. Each scenario has only one data-bsa-appendix visible at a time (others are commented out below). To run a scenario, uncomment its block and refresh.
  5. Verify the appendix lands in the expected spot per the comment.

Note: the script's querySelector picks the FIRST data-bsa-appendix in the document; only one scenario should be active per page load.
